关于在omnet++中添加inet framework框架

2013-08-11
畅游互联网
inet, makefile, omnet++

inet framework 支持很多协议在omnet++上的仿真

inet详细内容及下载地址见wiki百科，

下载后将inet解压，并把inet整个文件夹复制到omne++安装下面，

然后在omnet++控制台下输入如下命令

　　 make makefile

　　make 需要一些时间，耐心等待

　 进入examples目录下　输入命令./rundemo 就可以看到那上面的例子

当然在omnet++下import这些example 按常规导入，找到inet目录即可

下面你可以在你的工程里引用inet里面的inet工程，如下：

  • File -> New -> Omnet++ Project for C++
  • enter a name and confirm
  • open project properties of your new project
  • enable the INET checkbox in the project references and confirm
  • use all the nice INET stuff in your project
