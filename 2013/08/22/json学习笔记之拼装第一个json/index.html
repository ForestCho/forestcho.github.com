JSON学习笔记之拼装第一个json

2013-08-22
畅游互联网
JSONObject, XML, json

实习将近一个月了，时间过的有点冲动，实习中除开不断的要把数据经过处理传送到前段，也要讲一下不同格式的数据经过处理存入数据库，而用到的数据格式大多是JSON(JavaScript Object Notation)和XML(extensible markup language,)，这两者各有优劣，XML很适合人去理解，而JSON相对比较费解，但这跟二进制有点类似，人难理解的东西机器则容易阅读。
XML天生有很好的扩展性，JSON当然也有，没有什么是XML能扩展，而JSON却不能扩展的。不过JSON在Javascript中很适合，可以存储Javascript复合对象，有着xml不可比拟的优势。公司之前的数据传输多是XML格式，现在基本上都是用JSON格式，相比这跟手机app应用中传输的数据简单性，如果非要用XML则显得“大材小用”，反而解析起来相对困难。
